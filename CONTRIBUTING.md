# Contributing to CanvasFlow

## Workflow

We follow **GitHub Flow** with conventional commits and branch protection.

### Branch naming

Format: `<type>/<short-description>`

Examples:
- `feat/auth-jwt-rotation`
- `fix/cursor-jitter`
- `chore/upgrade-nextjs`
- `docs/api-reference`
- `refactor/extract-renderer`
